2. Click "New Application"
3. Go to "Bot" section
4. Click "Add Bot"
5. Enable these Privileged Gateway Intents:
   - âœ… Presence Intent
   - âœ… Server Members Intent
   - âœ… Message Content Intent
6. Click "Reset Token" and copy it
7. Paste it in your .env file as DISCORD_TOKEN

## ğŸ”— Inviting Your Bot

Use this URL (replace YOUR_CLIENT_ID):
```
https://discord.com/api/oauth2/authorize?client_id=YOUR_CLIENT_ID&permissions=8&scope=bot%20applications.commands
```

Get YOUR_CLIENT_ID from the "General Information" tab in Discord Developer Portal.

## ğŸ†˜ Troubleshooting

**Bot not starting?**
- Check .env file has correct values
- Verify MySQL is running
- Look at logs/error.log

**Commands not working?**
- Run `/setup` first
- Wait 1-2 minutes for Discord to sync commands
- Check bot has proper permissions

**Database errors?**
- Verify MySQL is running
- Check database exists: `SHOW DATABASES;`
- Verify credentials in .env
