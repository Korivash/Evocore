## 🎯 Quick Start Workflow

### 1. Initial Setup
```bash
# Extract files
cd C:\DiscordBot\

# Install dependencies
npm install

# Configure .env file
copy .env.example .env
# Edit .env with your credentials
```

### 2. Deploy Commands
```bash
# For testing - deploy to your test server (instant)
npm run deploy-guild 123456789012345678

# For production - deploy globally (takes up to 1 hour)
npm run deploy
```

### 3. Start Bot
```bash
# Development
npm start

# Production (with PM2)
pm2 start index.js --name discord-bot
pm2 save
```

### 4. Configure in Discord
```
/setup              # Initial configuration
/config view        # Check settings
/config automod     # Enable auto-moderation
```

### Development Workflow:
```bash

# 2. Test immediately in your dev server:
npm run deploy-guild 123456789012345678

# 3. Commands appear instantly, test thoroughly
# 4. Once satisfied, deploy globally:
npm run deploy

# 5. Bot continues running the whole time!
```

### Production Updates:
```bash
# 1. Stop bot (if needed for code changes)
pm2 stop discord-bot

# 2. Update files
# 3. Deploy commands
npm run deploy

# 4. Restart bot
pm2 restart discord-bot

# 5. Commands propagate within 1 hour globally
```

### Emergency Command Clearing:
```bash
# If commands are messed up:
npm run clear-commands

# Wait a moment, then re-deploy:
npm run deploy
```
